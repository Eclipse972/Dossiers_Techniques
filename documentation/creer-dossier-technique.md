# Les dossiers sur le site
Liste des dossiers où intervenir
- public/supports/ : contient les fichiers asociés au support (images, pièces, assemblage, mises en plen, ...)
- src/Vue/supports : templates twig
- src/Controleur : l controleur générant toutes les pages
- scr/config/routes.php : contient toutes les routes des pages du dossier technique

IDÉE: script qui crée les dossiers ainsi que les ébauches des scripts (controleur, templates)
